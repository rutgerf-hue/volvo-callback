Homey Volvo integration — personal home-automation panel. Last updated: 22 May 2026.

1. About this application

This application is a personal, non-commercial home-automation panel. It uses the Volvo Cars Connected Vehicle and Location APIs to display information about a single vehicle owned by the user (the operator of the panel) on the user's own self-hosted dashboard.

2. Scope of use

  • The application is intended for use by a single individual (the owner of the Volvo Cars account that grants authorisation) with that individual's own vehicle(s).
  • The application is not offered as a service to third parties.
  • No fees are charged and no commercial activity is conducted through the application.

3. Data handling

  • Data retrieved from the Volvo Cars APIs (fuel level, odometer, door / window state, location, diagnostics, etc.) is displayed only on the user's own self-hosted panel.
  • No data is shared with any third party.
  • No data is published, broadcast, or sold.
  • Access tokens and the VCC API key are stored on the user's own machine and are never transmitted to anyone other than Volvo Cars' authorisation server.
  • The user may revoke access at any time by deleting their authorisation in their Volvo ID account, at which point this application will no longer be able to retrieve vehicle data.
